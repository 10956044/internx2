# InternX - 實習機會整合平台

## 專案概述
InternX 是一個專注於大學生實習機會的整合平台，旨在為學生提供便捷的實習資訊查詢、經驗分享和職涯諮詢服務。平台整合了各大企業的實習機會，並提供專業的職涯規劃建議，幫助學生找到理想的實習機會。

## 專案目標
1. 建立完整的實習機會資料庫
2. 提供直觀的搜尋和篩選功能
3. 促進實習經驗的分享與交流
4. 提供專業的職涯諮詢服務
5. 建立企業與學生之間的橋樑
